Welcome to the kdtreeGeocoder wiki!

This is'nt my original creation but a modification and improvement on the project already done at AReallyGoodName

Lets see how the library actually works.

Put the NP.zip file into the android projects assets folder. This file will be used as the database for the cities of Nepal.

Then initialize the ReverseGeoCoder class with

/* This will initialize the reverse geocoder class with the NP.zip file from the assets and the "true" tells the geocoder to get the major cities only */ ReverseGeoCoder.initialize(getActivity().getAssets(),"NP.zip",true);

//from this point onward you can call the instance of the ReverseGeoCoder //so to call the geocoder

GeoName geoName = ReverseGeoCoder.getInstance().nearestPlace(25.66,85.77);

//then the name of the city can be fetch from String address = geoName.name+", "+geoName.country;

This is the basic way of how the code actually works in the original library.

The modification comes here

You can use KDTreeGeoCoder class that will run a async task for you and do all the converting tasks, providing you with a clean dialog and a string address.

For that,

ReverseGeoCoder.initialize(getActivity().getAssets(),"NP.zip",true);

KDTreeGeoCoder coderTask = new KDTreeGeoCoder(getApplicationContext(),1234,true,new OnGeoCodeCompleteListener(){ public void onGeoCodeComplete(int requestCode, String geoName){ //do work with the geoName and address here according to the requestCode } `});
